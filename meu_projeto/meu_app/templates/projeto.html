Características do Projeto

Aplicação Desktop

-> O sistema principal do projeto foi realizado na aplicação desktop, desenvolvido em Python com aplicação visual em PyQt5. O aplicativo inclui um sistema completo de cadastro de veiculos e usuários, vizualização de vagas disponíveis, além de outros tipos de serviços e dentre outras aplicações.

COMO FOI ESTRUTURADA?

  • Modelo MVC (Model - View - Controller).
  • Possui um sistema de autenticacao completo.
  • Querys de SQL do Banco de Dados para automatização.
  • Link para o repositório: github.com/DashVision/gurgel-park-desktop

ALGUMAS BIBLIOTECAS USADAS:

  • BRCRYPT - Usada para criptografar senhas de forma segura
  • DOTENV - Usada para carregar variáveis de ambiente de um arquivo '.env' permitindo guardar informações sensíveis ou de configuração fora do código
  • PYDANTIC - Criar modelos de dados estruturados

Aplicação Web

-> A aplicação web demonstrativa foi desenvolvida através do framework Django com o proposito de integrar o backend e o frontend sem complicações, além disso o site foi estrutura e estilizado em HTML5 e CSS3

-> O propósito do Django foi desenvolver um processo de autenticacao de Cadastro e Login, permitindo o acesso ao site apenas para quem fosse registrado.

Estruturação

  • Sistema de rotas (URLS) - '127.0.0:1800/auth/projeto/.
  • Métodos Users e Autenticate.
  • Respostas em HttpResponse.
  • Métodos GET e POST.

Propósito

  • Exemplos de aplicação do DJANGO
